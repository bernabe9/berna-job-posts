## Company Overview

AirOps is a content engineering platform helping marketers maintain brand visibility in the AI era. The company is backed by prominent VCs including Greylock and Unusual Ventures, with offices in San Francisco, New York, and Montevideo.

## Key Responsibilities

- Build and maintain integrations between AirOps and customer content management systems like WordPress and Contentful
- Translate customer requirements into technical specifications
- Implement REST, JSON, and XML API integrations with third-party platforms
- Create documentation and setup guides for technical enablement
- Develop reusable integration frameworks
- Monitor integration health and implement error handling
- Collaborate across Product, Engineering, and Customer Success teams

## Required Qualifications

- 3+ years building production software and API integrations
- Strong JavaScript and Python proficiency
- REST API design and implementation experience
- Modern web development knowledge
- Problem-solving abilities and debugging expertise
- Ability to manage multiple projects independently
- Comfort navigating ambiguity in fast-paced environments

## Nice-to-Have Skills

- CMS integration experience (WordPress, Contentful, Webflow)
- Familiarity with task management APIs (Asana, Jira, ClickUp)
- Webhook and event-driven system knowledge
- Generative AI and content marketing workflow experience
