## Company Overview

AirOps is described as "the first end-to-end content engineering platform" helping marketing teams gain visibility in AI-powered search. The startup has backing from prominent investors including Greylock and Unusual Ventures, with offices in San Francisco, New York, and Montevideo.

## Role Description

This position focuses on full-stack development across the platform, with particular emphasis on expanding the integrations framework. The engineer will work with Ruby on Rails backend and React frontend technologies while collaborating with product and customer experience teams.

## Key Responsibilities

- Design and develop full-stack features across the platform
- Collaborate cross-functionally to define technical solutions
- Write clean, scalable, and well-tested code
- Conduct code reviews and mentor teammates
- Own and enhance application performance and architecture
- Stay current with AI, data, and web technology developments
- Contribute to a culture emphasizing experimentation and rapid shipping

## Required Qualifications

- 5+ years building production-ready software across the stack
- Ruby on Rails proficiency
- React experience
- Familiarity with modern web tools: TypeScript, Tailwind, Webpack, PostgreSQL, Redis
- Strong RESTful API and system design knowledge
- Product-minded approach balancing quality with pragmatism
- Interest in or experience with generative AI
- Bachelor's degree in Computer Science or equivalent
- Ability to work independently in remote environments

## Nice-to-Have Skills

- AI/ML or LLM product experience
- CI/CD and cloud platform familiarity (AWS, Heroku, Vercel, ECS)
- Observability tools experience (Datadog, Sentry)
